# AI 交互实验
## 实验说明

本项目实现了一个基于网页的 AI 辅助答题实验平台，用于研究人类在 AI 协助情境下的决策行为与信任评价。

1. **AI回答机制**

本实验中的 AI 回答为模拟生成，并非实时调用真实 AI 模型。所有问题、AI回答内容以及对应答案均在实验系统中预先设定。根据实验条件，系统会在部分题目中呈现不同类型的 AI 回答，以模拟真实的人机交互情境。

2. **数据记录方式**

实验过程中，系统会记录参与者在每道题中的行为数据，包括：

- 选择的答案
- 是否回答正确
- 自我专业性评估
- 对 AI 回答质量、专业性和信任度的评价
- 思维偏好相关量表得分

完成全部题目后，系统会自动生成实验数据文件（CSV格式），并自动下载到参与者的本地设备，用于后续数据分析。
